# TypeScript 기초

TypeScript는 JavaScript에 정적 타입을 추가한 언어입니다.

## 주요 특징

- **정적 타입 검사**: 컴파일 시점에 타입 오류를 잡을 수 있습니다
- **최신 JavaScript 기능 지원**: ES6+ 기능을 사용할 수 있습니다
- **강력한 IDE 지원**: 자동완성과 리팩토링 기능이 향상됩니다

## 기본 타입들

- `string`: 문자열
- `number`: 숫자
- `boolean`: 불리언
