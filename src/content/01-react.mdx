## 주요 특징

- **컴포넌트 기반**: 재사용 가능한 UI 컴포넌트를 만들 수 있습니다
- **가상 DOM**: 효율적인 렌더링을 제공합니다
- **단방향 데이터 흐름**: 예측 가능한 앱 상태 관리가 가능합니다

## 기본 개념

- JSX: JavaScript XML
- Props: 컴포넌트 간 데이터 전달
- State: 컴포넌트 내부 상태 관리
